    ¿Qué es el HTML exactamente?

    HTML es un lenguaje de marcado que sirve para describir de una manera estructurada mediante etiquetas el contenido de un documento. Esto tiene la gran ventaja de que podemos añadir con ello semántica (significado) al documento que una máquina puede entender. link
